# Scout Plan Build Test

## Overview

Run Scout, Plan, Build, then Test Features.

## Inputs

- Feature description
- Optional URL for runtime testing

## Workflow

1. Run Scout.
2. Run Plan.
3. Run Build.
4. Run Test Features (or project tests).

## Output

- Plan file path, implementation summary, test results
